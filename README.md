## Requirements

* CakePHP 2.0 or greater.
* PHP 5.3.10 or greater.
* PhpRedis.

## Installation

* Clone/Copy the files in this directory into `app/Plugin/Redis`

## Configuration

* Ensure the plugin is loaded in `app/Config/bootstrap.php` by calling:

```
CakePlugin::load('Redis');
```
